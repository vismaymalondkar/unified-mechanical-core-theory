We don't need "new physics"; we just need to fix the leaks.

## ⚙️ Core Architecture (The Axioms)

To run this universe, you only need to accept two foundational variables:

1.  **The Absolute Medium:** Space is not empty. It is a physical, high-density ($\rho \approx 10^{11}$) acoustic superfluid. It exists under a constant, crushing baseline static pressure ($P_0$).
2.  **The Standard Pipe:** There are no "fundamental particles." Matter is simply the **Standard Universal Pipe**—a fluid vortex tube—tied into various 3D topological knots.

From these two rules, all "forces" emerge as simple plumbing consequences:

  * **Mass:** Hydrodynamic Drag ($A_{eff}$).
  * **Charge:** Volumetric Flow Rate ($Q$).
  * **Gravity:** A Bernoulli Pressure Sink.
